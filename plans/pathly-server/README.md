# pathly-server — Plan Index

Browser-based dashboard for the Pathly CLI. Adds `pathly server` to start a
FastAPI backend + React frontend that visualises feature pipeline state in real
time.

## Story IDs at a glance

| ID   | Summary                                              |
|------|------------------------------------------------------|
| S1   | `pathly server` starts dashboard, opens browser      |
| S2   | Kanban board shows PLAN → BUILD → REVIEW → TEST → DONE |
| S3   | Conversation cards show TODO / IN_PROGRESS / DONE    |
| S4   | Sidebar MD file list, click to preview rendered markdown |
| S5   | Chat "what's next?" grounded in STATE.json           |
| S6   | [Go] / [Pause] buttons trigger CLI actions           |
| S7   | Agent status indicators (running / idle) in sidebar  |
| S8   | Board updates in real time as EVENTS.jsonl changes   |
